# Loja Online de Livros the bookshelf

Projeto em Typescript, aplicando boas práticas de desenvolvimento e arquitetura. 
A idéia aqui era fazer as coisas, então, eu evitei o uso de libraries que facilitassem minha vida.

É um projeto simples que permite o usuário navegar pelos livros, comentar (ainda não), avaliar, comprar e gerenciar o carrinho de compras.

Integrado com meu backend em Express (ainda não).
