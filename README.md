# Loja Online de Livros the bookshelf ([disponível aqui](https://thebookshelf-wheat.vercel.app))

Esse é o frontend do projeto, o backend pode ser encontrado [nesse link](https://github.com/starzlocker/library_management_system_backend).

Projeto em Typescript e React, aplicando boas práticas de desenvolvimento e arquitetura. 
A idéia aqui era fazer as coisas, então eu evitei o uso de libraries que facilitassem minha vida.

É um projeto simples que permite o usuário navegar pelos livros, comentar (ainda não), avaliar, comprar e gerenciar o carrinho de compras.
